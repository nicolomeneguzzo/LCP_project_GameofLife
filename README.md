# LCP_project_GameofLife

Repository per il progetto finale di LCP mod.A 

Gitinstruction:
Un po di comandi e istruzioni comode che mi dimentico sempre.

set up:
ognuno si crei un branch in cui lavorare, dentro al branch develope usare il comando
`git switch -c nome_branch`
`git branch --set-upstream-to=origin/develop develop`

git routine:

nel branch develope per aggiornare il branch locale
`git pull`

aggiornare il prorpio branch locale
`git switch nome_branch` o `git checkout nome_branch`
`git fetch
`git merge develop` 


ora puoi lavorare tranquillo :)
